Operator Aritmatika--

terdapat 4 jenis operator utama dalam Operator Matematika Javascript diantarnya yaitu:

  • Penjumlahan (+)
  • Pengurangan (-)
  • Perkalian (*)
  • Pembagian (/)

selain di operator diatas terdapat operator lainnya yang disebut sebagai modulu/modulus/sisa bagi
dengan simbol %

Operator Precedence

Operator Precende adalah urutan pengoperasian dalam Matematika. Operator dimulai dari urutan KuKaBaTaKu yaitu Kurung (), Kali *, Bagi /, Tambah +, dan Kurang -

Operator Penugasan

Operator Penugasan adalah operator yang digunakan untuk memberikan nilai pada variabel. Operator ini menggantikan nilai dari variabel sesuai dengan nilai yang kita simpan/masukan.

Jenis Operator Penugasan

  • Penjumlahan (+=)
  • Pengurangan (-=)
  • Perkalian (*=)
  • Pembagian (/=)
  • Modulus (%=)

Contoh Operator Penugasan

  • Contoh Penjumlahan: x += 5;
  • Contoh Pengurangan: y -= 3;
  • Contoh Perkalian: z *= 2;
  • Contoh Pembagian: a /= 4;
  • Contoh Modulus: b %= 7;

Operator Perbandingan

Operator Perbandingan adalah operator yang digunakan untuk membandingkan dua buah nilai. Operator ini akan menghasilkan nilai boolean yaitu true atau false.

Jenis Operator Perbandingan

  • Lebih Besar (>)
  • Lebih Kecil (<)
  • Lebih Besar Sama Dengan (>=)
  • Lebih Kecil Sama Dengan (<=)
  • Sama Dengan (==)
  • Tidak Sama Dengan (!=)

Contoh Operator Perbandingan

  • Contoh Lebih Besar: x > 5;
  • Contoh Lebih Kecil: y < 3;
  • Contoh Lebih Besar Sama Dengan: z >= 2;
  • Contoh Lebih Kecil Sama Dengan: a <= 4;
  • Contoh Sama Dengan: b == 7;
  • Contoh Tidak Sama Dengan: c != 8;
