PENGKONDISIAN | ELSE IF

Else If adalah keyword yang digunakan untuk menambahkan kondisi (lainnya) pada percabangan atau pengkondisian.

Contoh Penggunaan Else IF

        
            var angka = prompt('Masukan Angka =');
            if ( angka % 2 === 0 ) {
                alert( angka + 'adalah bilangan Genap');
            } else if ( angka % 2 === 1 ) {
                alert ( angka + 'adalah bilangan Ganjil');
            } else {
                alert(' Masukan bukanlah Angka!!');
            }
