Operator Logika

Operator Logika adalah operator yang digunakan untuk menentukan dari berbagai ekspersi logika

  • && (AND)
  • || (OR)
  • ! (NOT)
  • (5 > 3) && (10 < 20)
  • (10 == 10) || (5 > 8)
  • !(7 < 4)

Operator String

Operator String adalah operator yang digunakan untuk menggabungkan dua buah string

  • +
  • "Hello" + "World"

Operator TypeOf

Operator TypeOf adalah operator yang digunakan untuk mengetahui tipe data dari suatu variabel

  • typeof 10
  • typeof "Hello"
  • typeof true
  • number
  • string
  • boolean

Operator Kondisional

Operator Kondisional adalah operator yang digunakan untuk menentukan ekspresi yang akan dieksekusi berdasarkan kondisi tertentu

  • condition ? expression1 : expression2
  • let age = 20; let status = age >= 18 ? "Adult" : "Underage";
