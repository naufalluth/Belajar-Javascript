Tipe Data: String

String adalah tipe data dalam javascript yang digunakan untuk merepresentasikan data tekstual atau plain text.

String di javascript bisa ditulis dengan menggunakan tanda kutip satu ('...') atau tanda kutip dua ("...").

Escape Character

Escape character adalah karakter khusus yang digunakan untuk merepresentasikan karakter yang tidak bisa ditulis langsung dalam string. Escape character ditulis dengan menggunakan tanda backslash (\).

Berikut adalah beberapa escape character yang sering digunakan:

  • \n : newline
  • \t : tab
  • \b : backspace
  • \r : carriage return
  • \f : form feed
  • \\ : backslash
  • \' : single quote
  • \" : double quote

Contoh Penggunaan

Contoh penggunaan escape character:
var alamat = "Jl. Jend. Sudirman\nNo. 123";
var nama = "John Doe";
var quote = "He said, \"Hello, World!\"";

Contoh Operasi pada String

Contoh penggunaan tipe data string:
var nama = "John Doe";
var alamat = 'Jl. Jend. Sudirman No. 123';

String juga bisa digabungkan dengan menggunakan operator (+).
var nama = "John";
var belakang = "Doe";
var namaLengkap = nama + " " + belakang;

String juga bisa diakses dengan menggunakan indeks karakter.
var nama = "John Doe";
var karakterPertama = nama[0];
var karakterTerakhir = nama[nama.length - 1];

String juga bisa diakses dengan menggunakan method-method yang disediakan oleh javascript.
var nama = "John Doe";
var hurufKapital = nama.toUpperCase();
var hurufKecil = nama.toLowerCase();
var panjang = nama.length;
var index = nama.indexOf("Doe");
var potongan = nama.slice(5, 8);
